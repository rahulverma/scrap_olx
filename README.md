Olx Scraper
=========

The idea is to crawl [Olx] and generate stats that are otherwise unavailable.
Stores the crawled data in an sqlite database.


Installation
--------------
Needs only [Scrapy] to be available


Running
--------------

```sh
scrapy crawl olx
```

For visualizing the data there is [olx_front].
